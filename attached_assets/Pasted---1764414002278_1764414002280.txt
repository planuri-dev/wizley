제공해주신 구체적인 정보(핵심 서비스, 타겟, 벤치마킹 사이트)를 바탕으로 설계를 대폭 정교화했습니다.

특히 \*\*'플랜유리'\*\*와 **'올댓워시'** 같은 경쟁사 사이트는 \*\*"신뢰감"\*\*과 \*\*"솔루션의 시각화(하드웨어+소프트웨어)"\*\*가 핵심입니다. 이를 반영하여 AI가 단순히 정보만 나열하는 것이 아니라, **잠재 고객(창업 희망자)의 문의(DB)를 확보하는 데 최적화된 구조**로 코드를 짜도록 유도하겠습니다.

수정된 3종 세트입니다.

## **문서 1. [무인세차 통합 솔루션] 요구사항 정의서 (PRD.md)**

### 프로젝트 개요

\*\*하드웨어(키오스크)\*\*와 **소프트웨어(SaaS)**, **구축 컨설팅**을 아우르는 무인세차 토탈 솔루션 기업의 공식 웹사이트. 벤치마킹 사이트 수준의 전문성을 갖추되, 예비 창업자의 \*\*상담 신청(Lead Generation)\*\*을 최우선 목표로 함.

### 핵심 타겟 (페르소나 정의)

1.  **예비 창업자:** "무인 세차장이 돈이 될까?"를 고민하며 안정적인 수익 모델과 쉬운 운영 시스템을 찾는 퇴직자나 투자자.
2.  **기존 세차장 운영자:** "인건비를 줄이고 싶다"는 니즈를 가진, 시스템 무인화를 원하는 사장님.

### 주요 기능 (MVP Scope)

| 기능 | 설명 (AI 구현 가이드) |
| :--- | :--- |
| **(필수 기능 1) 3대 솔루션 쇼케이스** | **1. 하드웨어(키오스크/결제기)**, **2. 운영 플랫폼(SaaS/앱)**, **3. 구축 컨설팅**을 탭(Tab) 또는 그리드 형태로 시각화하여 소개. (제품 스펙 및 특장점 강조) |
| **(필수 기능 2) 성공 사례(포트폴리오)** | 실제 설치된 세차장 사진과 함께 '구축 지역', '도입 시스템' 등을 카드 형태로 나열. (신뢰도 상승 장치) |
| **(필수 기능 3) 스마트 상담 신청 폼** | 단순 문의를 넘어 **창업 희망 지역**, **보유 부지 평수**, **연락처**를 입력받는 맞춤형 폼. (접수 시 관리자 이메일로 알림 전송) |
| **그 외 기능** | 회사 연혁, 언론 보도, 세부 견적 계산기 등은 MVP 이후 고도화. |

### 사용자 흐름 (User Flow)

1.  **접속:** 메인 배너에서 '무인 세차 창업의 모든 것' 등의 문구와 하드웨어 이미지를 확인
2.  **설득:** 스크롤을 내리며 [하드웨어 스펙] -\> [운영 SaaS 화면] -\> [성공 사례] 순으로 정보 습득
3.  **전환:** 페이지 하단 또는 우측 고정 버튼(Floating Button)을 통해 [무료 창업 상담 신청] 완료

## **문서 2. [무인세차 통합 솔루션] 기술 사양서 (TRD.md)**

### 권장 스택

| 구분 | 스택 | 버전 |
| :--- | :--- | :--- |
| **프레임워크** | **Next.js** (App Router) | 14.x (Stable) |
| **언어** | **TypeScript** | 5.x |
| **UI 라이브러리** | **Shadcn/ui** + **Tailwind CSS** | 최신 버전 |
| **폼 처리** | **React Hook Form** + **Zod** | 최신 버전 |

### 선정 이유

  * **Shadcn/ui 추천:** '적당한 디자인' 이상을 구현하기 위해 선정했습니다. 벤치마킹 사이트처럼 깔끔한 **카드 UI, 탭 전환, 아코디언(Q\&A)** 등을 AI가 가장 빠르고 예쁘게 만들 수 있는 최신 라이브러리입니다.
  * **Next.js:** 많은 이미지(키오스크, 세차장 전경)를 다루므로 **이미지 최적화(Image Optimization)** 기능이 내장된 Next.js가 필수입니다.
  * **Zod:** 상담 신청 시 전화번호 형식, 필수 입력값 등을 AI가 엄격하게 검증하여 \*\*유효한 고객 데이터(Lead)\*\*만 받도록 합니다.

### 핵심 아키텍처 & 스타일 가이드

#### 1\. 디렉토리 구조 (App Router)

```
/app
├── / (Landing Page: 모든 섹션을 스크롤 형태로 배치 권장)
├── /solutions (Detail: 하드웨어/SaaS 상세)
├── /portfolio (Detail: 설치 사례 리스트)
├── /contact (상담 신청 전용 페이지)
├── layout.tsx (헤더: 로고/네비게이션, 푸터: 회사정보 포함)
/components
├── /sections (Hero, Features, Portfolio, Contact 섹션 분리)
├── /ui (Shadcn 컴포넌트 자동 생성 폴더)
```

#### 2\. 디자인 가이드 (벤치마킹 반영)

  * **메인 컬러:** 신뢰를 주는 **Navy(남색)** 또는 **Deep Blue**를 기조로 하고, 강조색(버튼)은 **Orange**나 **Sky Blue** 사용.
  * **레이아웃:** 텍스트보다는 **이미지와 아이콘 위주**의 레이아웃 구성. (예: "24시간 무인 관제" 옆에 CCTV 아이콘 배치)

### AI 코딩 주의사항

  * **이미지 처리:** 모든 이미지는 Next.js의 `<Image />` 컴포넌트를 사용하고 `placeholder="blur"` 옵션을 적용할 것.
  * **반응형:** 모바일(폰)에서도 상담 신청 폼 입력이 편하도록, 입력 필드 크기와 간격을 넉넉하게 잡을 것.

## **문서 3. AI 코딩 착수용 프롬프트 (Tasks.md)**

아래 내용을 복사해서 AI 코더에게 전달하세요.

```markdown
"너는 **Next.js 14 (App Router)**와 **Shadcn/ui**를 마스터한 수석 웹 개발자야."
"이번 프로젝트는 무인세차 솔루션(키오스크, SaaS, 컨설팅)을 판매하는 기업 홈페이지 구축이야."

**[지시사항]**
1. 첨부한 **[요구사항 정의서]**를 바탕으로, 특히 **'예비 창업자'가 신뢰를 느낄 수 있는 구조**로 페이지를 구성해.
2. **[기술 사양서]**에 명시된 대로 **Next.js 14, Tailwind CSS, Shadcn/ui**를 반드시 사용해.
3. 디자인은 'Planuri'나 'Allthatwash' 같은 사이트처럼 **전문적이고 깔끔한 B2B 스타일**을 지향해. 메인 컬러는 깊은 네이비(Navy) 색상을 추천해.
4. **핵심 기능 구현:**
    - **Hero Section:** 임팩트 있는 문구와 배경 이미지.
    - **Solution Section:** 하드웨어, SaaS, 컨설팅을 3개의 카드 또는 탭으로 구분해서 보여줘.
    - **Inquiry Form:** 이름, 연락처, 창업희망지역, 내용을 입력받는 폼을 만들고 Zod로 유효성을 검사해.

5. **작업 순서:**
    - 먼저 `npx create-next-app@latest` 기반의 폴더 구조와 필요한 `shadcn` 컴포넌트(Button, Card, Input, Form 등) 설치 명령어를 알려줘.
    - 그 다음, 메인 페이지(`page.tsx`)의 레이아웃을 잡아줘.

이제 프로젝트 폴더 구조부터 제안해 줘.
```
